  • Complexity definititions/calculations

    Each simple operation (+, -, *, /, =, if, function call) takes one time step/unit.

    T(n) = the number of steps

    We are not interested in the exact number of steps for a given algorithm, we are interested in its order of growth (i.e., how does the number of steps change if the value of n increases)

    O notation

    The O-notation provides an asymptotic upper bound for a function: for all values of n (to the right of n0) the value of the function f (n) is on or below c · g(n)

    f(n) ∈ O(g(n)) if lim n→∞ f(n)/g(n) is 0 or a constant (but not ∞)

    Ω-notation

    The Ω-notation provides an asymptotic lower bound for a function: for all values of n (to the right of n0) the value of the function f (n) is on or above c · g(n)

    f(n) ∈ Ω(g(n)) if lim n→∞ f(n)/g(n) is ∞ or a nonzero constant.
